# Tier 6 Squad Intelligence

Tier 6 turns Sergeant into a squad-led reviewer.

Sergeant remains the final authority. Specialist agents produce focused reports, then Sergeant weighs the evidence and issues one command summary.

## Squad

- Quartermaster — shared review context, indexes, cached evidence
- Scout — repository scan, changed-file map, dependency signals
- Engineer — architecture, API contracts, cross-file impact
- Medic — security and taint analysis
- Mechanic — performance, blocking calls, concurrency/race signals
- Analyst — root-cause grouping, priority, review intelligence
- Challenger — second-opinion comparison and disagreement pressure test
- Archivist — verified learning and memory candidates
- Judge — graduation benchmark and reference comparison
- Hermes — GitHub/app messenger and delivery agent

## Rule

Specialists advise. Sergeant commands the final review.

The squad increases coverage without fragmenting authority.
