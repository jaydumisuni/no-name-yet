# Sergeant Battle Tests

This folder records real public pull requests used to evaluate Sergeant.

Each fixture should include:

- repository and pull request link
- pull request outcome
- changed files
- reviewer feedback
- expected comparison target
- what a strong review should catch

## Fixtures

- `psf-requests-7502.md` - merged Requests PR with review discussion about test clarity and focused coverage.
- `pallets-flask-5812.md` - merged Flask architecture PR covering context lifecycle, docs, compatibility, and later follow-up evidence.

## Rule

Sergeant ranking should be earned from repeatable evidence on real code review cases.
